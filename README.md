## 🎮 Overview

katamariio is a Forge mod for Minecraft 1.20.1. It lets you control a growing sphere that pulls in nearby blocks as you move through the world.

The goal is simple: start small, collect blocks, and get bigger. As the sphere grows, it can absorb more of the area around it. This makes each run feel different, since the sphere changes as you play.

Use it when you want a fast, block-collecting game mode inside Minecraft. It works on Windows and is built for players who want a simple install and quick start.

## 🧩 How It Works

katamariio centers on a single idea: a sphere that absorbs nearby blocks.

Here is the basic flow:

- You start as a small sphere.
- You move around the world and touch blocks.
- Nearby blocks get absorbed into the sphere.
- The sphere grows in size.
- A larger sphere can take in more blocks.

This gives you a clear loop with simple controls and steady progress. The more you collect, the more space you take up.
